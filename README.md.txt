# NIFTY 50 Forecasting using ARIMAX-EGARCH Model

## Project Overview
This project forecasts the NIFTY 50 index using ARIMAX, GARCH, and EGARCH models.

## Tasks Completed
- Data Collection using Python yfinance
- Exploratory Data Analysis (EDA)
- Data Visualization

## Tools Used
- Python
- Google Colab
- yfinance
- Pandas
- NumPy
- Matplotlib
- Statsmodels
- ARCH package

## Analysis Performed
- Correlation Analysis
- VIF Test
- Stationarity Tests (ADF, KPSS)
- CUSUM Test
- ACF/PACF Analysis
- ARIMAX Model
- GARCH Model
- EGARCH Model
- Forecast Accuracy Analysis

## Results
- Best Model: ARIMAX + EGARCH (Student's t)
- MAPE: 2.17%
